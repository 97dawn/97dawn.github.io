GPA_Calculator

Developing Period : 2018.01
Programming Language : Python
IDE : Eclipse

1. How to use

Once you run the application, you will see the dialog like the picture below.

You should select a school and enter a number of courses.

Then you will see the dialog with several blanks for credit number and comboboxes for letter grade.

After entering the credit number(s) and selecting the letter grade(s), once you press the Calculate button, you will get your GPA.
