About_Me_Chatbot

Developing Period : 2018.01
Programming Language : Java
IDE : Netbeans

1. Functions

This chatbot is a closed domain, retrieval-based model. It helps you to know Korean celebrities better. It shows the celebrity’s bio, instagram, twitter, and youtube videos.

2. How to use

Once you run the application, you will see the dialog like the picture below.

If you type a certain Korean celebrity’s name, you can check out his/her profile with his/her picture. Then, the chatbot will ask you about checking his/her SNS.

Exception) If you make a typo, the chatbot will say “I don’t know who this person is.” The chatbot doesn’t consider the capitalization.

Exception) If you don’t type anything for any chatbot’s question, the chatbot will say “Please write again.”

If you say ‘yes’ for checking his/her SNS, the chatbot will ask you to pick one of three types of SNS.

Exception) If you make a typo, the chatbot will say “I can’t understand.”

Let’s say you pick youtube. Then the chatbot will automatically show the youtube videos reagrding the celebrity.

Exception) If you pick the number which is not in the options, the chat will ask for write again.

If you say ‘yes’ to the question “Do you want to check another SNS?,” the chatbot will ask you to pick one type of SNS once again.

If you choose Instagram, then the chatbot will automatically show the the celebrity’s Instagram.

If you want to check another Korean celebrity, say no to the question “Do you want to check another SNS?”, say yes to the question “Do you want to search another Korean celebrity,” and type another celebrity’s name.

If the celebrity you choose doesn’t have the type of SNS you choose, the chatbot tells you that the celebrity doesn’t have that kind of SNS.

You can also stop looking for the celebrity’s SNS by typing ‘no’ to the question “Do you want to check another SNS?.” Also, if you don’t want to search celebrities any more, type ‘no’ to the question “Do you want to search another Korean celebrity?”
