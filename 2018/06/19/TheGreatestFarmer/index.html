The Greatest Farmer

Developing Period : 2018.05 - 2018.06 (1 month)
Tech : HTML, CSS, Bootstrap, JavaScript, JQuery, Ajax, PHP, MySQL, Apache, Visual Studio Code

Website Introduction

This is my CSE305(Principles of Database Systems) course team project website that helps buyers to buy farmers’ crops and users to learn how to grow crops from farmers’ posts. There are mainly two sections: Farmer section and Buyer section. Farmers can post their blog post, product post, and subscription post, save and read blog posts, and see the list of their products, subscriptions, and saved blog posts. Buyers can read and save blog posts, order and put products in their carts, subscribe subscriptions, see and write the comments on products and subscriptions, and see the list of products in their carts, ordered products, and saved blog posts.
